# IsingMERA
Julia implementation of binary 1D (uncapped) MERA quantum circuits with minimal bond dimension for ground-state preparation of the transverse-field Ising model.

The system size $L = 3 \times 2^T$ for PBC MERA, or equivalently the maximum correlation range $\xi = 3 \times 2^T$ for infinite OBC MERA, where $T$ is the number of MERA layers.
